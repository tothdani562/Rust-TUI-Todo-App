## 3) Kanban Lite (Todo / Doing / Done)
Rovid leiras: kartyak mozgatasa oszlopok kozott, prioritas cimkekkel.
Miert latvanyos: 3 paneles elrendezes, szines tagek, kijelolt kartya kiemeles.
Ajanlott technikak:
- Higher Order Functions (filter map collect)
- while let (event loop)
- Error handling (allapotfajl)
Extra: drag helyett billentyus mozgas.
